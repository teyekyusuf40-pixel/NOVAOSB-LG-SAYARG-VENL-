DOSYAYI İNDİR
Nasıl Kullanılır?
1️⃣ Python Yükleyin: Bilgisayarınızda Python yüklü olmalıdır. Eğer yoksa python.org üzerinden indirin.
2️⃣ Kütüphane Kurulumu: CMD'yi açın ve şu komutu yazın: pip install customtkinter
3️⃣ Çalıştırma: İndirdiğiniz NovaOS_Stealth.pyw dosyasına çift tıklayın. Siyah ekran açılmadan doğrudan arayüz gelecektir.
4️⃣ Yönetici İzni: Yazılım sistem ayarlarını (IP/DNS) değiştirdiği için açılışta çıkan "Yönetici olarak çalıştır" uyarısına "Evet" deyin.
